1.0 - Form Elements

Styling for all form related elements. Most of the stlying for these live in the objects/forms.styl file

1.1 - Text Inputs

When a text input is used on its own it will have 100% width. Using default styles for :focus

Default styling

 
<input type="text" name="name">
